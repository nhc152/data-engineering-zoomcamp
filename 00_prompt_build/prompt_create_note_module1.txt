[Tên file]
- Dạy về: [topic]
- Concepts chính: [list]

═══════════════════════════════
3. PHÂN TÍCH CODE FILES
═══════════════════════════════
Với mỗi file code, ghi rõ:

[docker-compose.yaml]
- Services: tên, image, ports, volumes, env vars, networks, depends_on
- Copy nguyên văn toàn bộ file

[Dockerfile]
- Base image, layers theo thứ tự
- Entry point, CMD
- Copy nguyên văn toàn bộ file

[ingest_data.py]
- Arguments nhận vào
- Logic chính từng bước
- Libraries dùng
- Copy nguyên văn toàn bộ file

[pyproject.toml]
- Dependencies list
- Copy nguyên văn toàn bộ file

[docker-pgadmin.sh / docker-postgres.sh]
- Làm gì, arguments, cách chạy
- Copy nguyên văn toàn bộ file

[terraform_basic/main.tf]
- Copy nguyên văn toàn bộ file

[terraform_with_variables/main.tf + variables.tf]
- Resources được tạo
- Variables: tên, type, default
- Copy nguyên văn toàn bộ file

[terraform_with_variable_AWS/main.tf + variables.tf + terraform.tfvars]
- Copy nguyên văn toàn bộ file
