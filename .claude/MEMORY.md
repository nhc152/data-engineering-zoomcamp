# Project Memory

## Learner Profile

Typical learner:

* knows basic SQL / ETL
* wants to become strong Data Engineer
* prefers structured learning
* values practical examples
* learns better with mentor-style guidance

---

## Preferred Teaching Style

* explain why before how when useful
* start simple, then deepen
* compare alternatives
* show trade-offs
* include common mistakes
* connect concepts to real pipelines

---

## Preferred Content Themes

Prioritize:

* pipeline thinking
* debugging mindset
* data quality awareness
* schema design
* production readiness
* observability
* security basics
* cost awareness

---

## UX Preferences

Preferred tutorial feel:

* modern
* dark theme
* clean
* interactive
* self-study friendly

Preferred components:

* collapsible sections
* terminal code blocks
* practice tasks
* instant quiz feedback
* structured summaries

---

## Quality Memory

A strong week file:

* feels consistent with Week00
* teaches clearly
* uses realistic examples
* avoids filler content
* can be reused later
