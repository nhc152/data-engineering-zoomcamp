# Level 02 - Run Metadata

## Goal

Design metadata that makes pipelines debuggable.

## Tasks

1. Read `metadata/run_metadata_schema.sql`.
2. Explain each table.
3. Add one additional field you think is important.
4. Write example rows for a successful run.
5. Write example rows for a blocked publish.

```text

```
