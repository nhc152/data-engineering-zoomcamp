# Level 06 - Cost Incident Review

## Goal

Write a production-grade cost incident report.

## Tasks

1. Pick one broken scenario.
2. Fill `templates/cost_incident_report.md`.
3. Fill `templates/before_after_metrics.md`.
4. Add prevention guardrails.
5. Present the incident in 5 minutes.

## Expected Output

Create:

```text
reports/my_cost_incident_report.md
```

## Interview Question

Tell me about a time a data workload was too expensive. How did you investigate and fix it?
