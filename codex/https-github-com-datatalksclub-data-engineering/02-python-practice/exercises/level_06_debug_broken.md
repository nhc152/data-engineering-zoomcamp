# Level 06 - Debug Broken Ingestion

Goal: debug intentional broken scenarios.

Scenarios:

1. `broken/01_pagination_bug.py`
2. `broken/02_retry_duplicate_load.py`
3. `broken/03_partial_file_write.py`
4. `broken/04_missing_env_var.md`
5. `broken/05_invalid_required_columns.md`

For each scenario:

1. State symptom.
2. Identify root cause.
3. Write diagnostic step.
4. Explain production impact.
5. Compare with matching solution.
