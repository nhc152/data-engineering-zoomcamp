# Level 05 - Incident Response

## Goal

Write incident reports that improve the system.

## Tasks

1. Pick one broken scenario.
2. Use `incidents/templates/incident_template.md`.
3. Write summary, impact, root cause, fix, prevention.
4. Add follow-up tasks.

## Expected Learning

An incident report is not blame. It is a reliability artifact.

## Deliverable

Create:

```text
incidents/my_incident_report.md
```
