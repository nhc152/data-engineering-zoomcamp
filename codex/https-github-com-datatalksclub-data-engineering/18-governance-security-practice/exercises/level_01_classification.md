# Level 01 - Data Classification

## Goal

Classify ecommerce data columns and decide default exposure.

## Tasks

1. Read `policies/data_classification_policy.md`.
2. Review `matrices/classification_matrix.md`.
3. Add 5 new columns to the matrix:
   - ip_address
   - device_id
   - coupon_code
   - gross_margin
   - shipping_address
4. Decide classification and default exposure.

## Expected Output

Create notes:

```text
notes/classification_review.md
```

Include reasoning for each column.
