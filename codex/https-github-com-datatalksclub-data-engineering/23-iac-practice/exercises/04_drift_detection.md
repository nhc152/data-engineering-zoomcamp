# Exercise 04 - Drift Detection

## Goal

Practice responding to infrastructure drift.

## Tasks

1. Read `broken/02_manual_drift.md`.
2. Read `docs/drift_response_runbook.md`.
3. Decide whether to update Terraform or revert infra.
4. Write a drift incident note.

## Expected Output

Your incident note should include:

- what changed
- who changed it if known
- risk
- decision
- prevention
