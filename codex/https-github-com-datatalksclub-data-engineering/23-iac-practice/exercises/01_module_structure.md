# Exercise 01 - Terraform Module Structure

## Goal

Understand why production IaC should be organized by modules and environments.

## Tasks

1. Inspect `terraform/modules/`.
2. Inspect `terraform/envs/dev`.
3. Inspect `terraform/envs/prod`.
4. Explain why dev/prod use same modules but different values.
5. Identify which resources are critical.

## Expected Output

Write notes:

```text
notes/module_structure_review.md
```

Include:

- why modules exist
- why env separation matters
- what should be different between dev and prod
